# 우테고 오픈미션 - 나만의 인터프리터 만들기

## 구현 목표

- [x] 문자열, 정수형 표현
- [x] 숫자 계산
- [ ] 함수 구현

## 기능 구현 사항

- 문자열 출력하기
- 정수 출력하기
- 정수 계산시 우선사항 적용하기
- 변수 구현하기
- 변수 타입 구현하기

## 목적

1. c++ 언어 학습
2. 인터프리터 언어와 컴파일러 언어의 차이 학습
3. 인터프리터 내부 구현 원리 이해

## 주요 클래스

1. Lexer: 소스 코드 문자열을 의미있는 논리 단위로 구분하는 역할. 토큰화
2. Expression : 소스 코드 실행.
3. BinaryExpr : +, - 같은 계산식을 실행하는 Expression
4. VariableExpr : 변수를 선언하는 Expression
5. Parser : 토큰화된 소스 코드에서 Expression을 생성하여 실제로 실행하는 역할
